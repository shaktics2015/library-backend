# Library Management REST API Server  

## To run App

    mvn spring-boot:run

## To run App with TestCases

    mvn -P runWithTest

`http://localhost:8080/api/swagger-ui/index.html?configUrl=/api/v3/api-docs/swagger-config#/BorrowHistory`
